Which Install Method Is Right for You?

There are four ways to install Trier OS. Read the descriptions below and pick the one that matches your situation. Then skip straight to that section — you don't need to read all four.

Most Popular 💾

EXE Installer

TrierOS-Setup-3.6.1.exe

Double-click to install. Works exactly like any other Windows program. Creates a Start Menu shortcut. Best for individual users and small teams.

→ Method A (Page 2)
🏢

MSI Installer

TrierOS-Setup-3.6.1.msi

For IT departments deploying Trier OS across multiple machines. Supports Group Policy, silent install flags, and enterprise deployment tools like SCCM.

→ Method B (Page 3)
📦

ZIP Portable

TrierOS-Setup-3.6.1.zip

No installer needed. Unzip and run. Great for servers, USB drives, air-gapped machines, or anywhere you can't run an installer.

→ Method C (Page 4)

From Source

Clone from GitHub

For developers, contributors, and Mac/Linux users. Requires Node.js and Git. Gives you the full codebase and the ability to customize everything.

→ Method D (Page 5)

Not sure? Pick EXE.

The EXE installer is the simplest path for 95% of users. You can always switch to a different method later — your data stays in the data/ folder and is portable between all install methods.

Before Any Install

Download the Installer Files

All installer files are on the GitHub Releases page. Here's how to get them:

1

Go to the Releases page

Open your browser and go to:

https://github.com/DougTrier/trier-os/releases/latest

This always takes you to the newest version.

2

Scroll down to "Assets" and download your file

Under the release notes you'll see an Assets section. Download the file for your method:

File NameUse For
TrierOS-Setup-3.6.1.exeEXE Installer (Method A)
TrierOS-Setup-3.6.1.msiMSI Installer (Method B)
TrierOS-Setup-3.6.1.zipZIP Portable (Method C)
⚠️
Windows SmartScreen may show a warning.

Because Trier OS is free and open-source (not code-signed with a paid certificate), Windows may say "unknown publisher." Click More info → Run anyway. This is normal for open-source software.

Method A — EXE Installer

Recommended for most users  ·  Installs like any Windows program  ·  ~5 minutes

1

Run the installer

Find the downloaded TrierOS-Setup-3.6.1.exe file (usually in your Downloads folder). Double-click it to start the installer.

If Windows shows a SmartScreen warning, click More info then Run anyway.

2

Follow the install wizard

Click through the installer steps:

① Accept the license agreement
② Choose your install folder (the default is fine)
③ Click Install
④ Click Finish when complete

3

Launch Trier OS

Look for the Trier OS icon on your Desktop or in the Start Menu. Double-click it to launch.

The app will open — on first launch it will set up the databases automatically. This takes about 30 seconds. Wait for the login screen to appear.

✅
Done!

Skip to the "First Login" section on Page 6 to log in.

Method B — MSI Installer

For IT departments  ·  Supports Group Policy & silent deployment  ·  ~5 minutes

1

Standard interactive install

Double-click TrierOS-Setup-3.6.1.msi to launch the Windows Installer wizard. Follow the prompts — same as any other MSI package.

2

Silent / automated install (IT use)

For scripted deployments, open a Command Prompt as Administrator and run:

# Silent install — no UI, installs to default location > msiexec /i TrierOS-Setup-3.6.1.msi /quiet /norestart # Silent install to a custom folder > msiexec /i TrierOS-Setup-3.6.1.msi /quiet INSTALLDIR="C:\TrierOS" # Uninstall silently > msiexec /x TrierOS-Setup-3.6.1.msi /quiet
3

Launch Trier OS

After install, launch from the Start Menu or run the TrierOS.exe from the install folder. On first launch, databases are set up automatically.

✅
Done!

Skip to the "First Login" section on Page 6 to log in.

Method C — ZIP Portable

No installer needed  ·  Run from any folder or USB drive  ·  ~5 minutes

💡
What "portable" means

Nothing gets installed on your computer. Trier OS runs directly from whatever folder you unzip it into. Move the folder to a USB drive and it runs from there too. Delete the folder to remove it completely — no uninstaller needed.

1

Unzip the file

Find TrierOS-Setup-3.6.1.zip in your Downloads folder. Right-click it and choose Extract All.

Choose a destination folder — for example C:\TrierOS\ or a folder on a USB drive. Click Extract.

2

Open the extracted folder

Navigate into the folder that was created. You should see a file called TrierOS.exe (or similar) inside.

3

Launch Trier OS

Double-click TrierOS.exe to launch. On first run, Trier OS will create a data/ folder in the same directory and set up the databases. This takes about 30 seconds.

⚠️
Don't move individual files out of the folder.

The portable version needs all its files together. You can move the entire folder anywhere, but don't rearrange what's inside it.

✅
Done!

Skip to the "First Login" section on Page 6 to log in.

Method D — From Source

Developers, contributors & Mac/Linux users  ·  Full codebase  ·  ~15 minutes

What You'll Need First

🟢
Node.js v22+

The runtime that powers Trier OS. Download from nodejs.org — get the LTS version.

Free — nodejs.org
🐙
Git

Used to download the code from GitHub.

Free — git-scm.com
🔍
Already have Node.js?

Open a terminal and run node --version. If you see v22 or higher, skip the Node.js install below.

1

Install Node.js (if needed)

Go to nodejs.org, click the big green LTS button, and run the installer. Accept all defaults. Make sure "Add to PATH" is checked.

Verify it worked:

$ node --version # Should show v22.x.x or higher
2

Install Git (if needed)

Go to git-scm.com/downloads, download for your OS, and run the installer with all defaults.

$ git --version # Should show git version 2.x.x
3

Download Trier OS

Open a terminal, navigate to where you want to put Trier OS, and run:

$ git clone https://github.com/DougTrier/trier-os.git trier-os $ cd trier-os
4

Install dependencies

This downloads all required packages. Takes 2–5 minutes.

$ npm install

Yellow warnings are normal. Only red errors need attention.

5

Set up configuration

Windows:

$ copy .env.example .env

Mac / Linux:

$ cp .env.example .env

The defaults in .env work out of the box for development. Trier OS auto-generates security keys on first boot.

6

Set up the database with demo data

$ npm run seed

Creates all databases and fills them with sample plants, assets, work orders, and user accounts. Takes about 30–60 seconds.

7

Start Trier OS

$ npm run dev:full

Wait for both of these lines to appear:

API | Trier OS server listening on port 3000 UI | Local: http://localhost:5173/

Then open your browser and go to http://localhost:5173.

💡
Keep the terminal open.

Trier OS runs as long as this terminal is open. Press Ctrl + C to stop. Run npm run dev:full again to restart.

Troubleshooting

Common Issues

?

Windows SmartScreen blocked the installer

Click More info on the SmartScreen dialog, then click Run anyway. This appears because Trier OS is free open-source software without a paid code-signing certificate. It is safe.

?

Login fails — "Invalid credentials"

Make sure the app finished its first-run database setup (wait for the login screen to fully appear). Check you're typing the username in all lowercase with no spaces. Passwords are case-sensitive.

?

App opens but shows a blank or loading screen

Wait 15–20 seconds on first launch — the database is being initialized. Refresh the page. If it persists, try closing and reopening the app.

?

"Port already in use" error (From Source only)

Another program is using port 3000 or 5173. Either close that program, or edit your .env file and change PORT=3000 to PORT=3001.

?

npm install fails (From Source only)

Confirm Node.js v22+ is installed: node --version. Delete the node_modules folder and run npm install again. Make sure you have an active internet connection.
